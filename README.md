# CopyCraft AI

- Editor con análisis SEO en tiempo real
- Reescritura de texto con múltiples tonos
- Auditoría completa de contenido

- 📁 Gestor de borradores local

- 🪄 Asistente IA para estructura y tono
- 📊 Análisis SEO detallado
